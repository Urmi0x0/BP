# SerialPort 串口属性赋值

关于SerialPort各个结点函数的解释

SerialPort() : 初始化 SerialPort 类的新实例。
PortName : 获取设置通信端口，包括但不限于所有可用的 COM 端口。
BaudRate : 获取或设置串行波特率。波特率数值越大，代表其传输速率越快。但是波特率并不是越高越好，一般情况下，波特率越高，其传播距离越短。对于使用何种波特率需要结合实际项目来分析选择。
DataBits ：获取或设置每个字节的标准数据位长度。一般来说，数据位越小，代表传播频率越高，传播内容更多。一般情况下我们选择 8 位。 ASCII 协议下可以选择 7 位。
Parity ：获取或设置奇偶校验检查协议。奇偶校验用于确认数据传输的准确性。本次选择了在 Modbus 协议中的 CRC校验 ，所以选择关闭奇偶校验。
StopBits : 获取或设置每个字节的标准停止位数。
ReceivedBytesThreshold : 获取或设置 DataReceived 事件发生前内部输入缓冲区中的字节数。
DataReceived : 指示已通过由 SerialPort 对象表示的端口接收了数据。
Open() : 打开一个新的串行端口连接。
Close() ：关闭端口连接，将 IsOpen 属性设置为 false ，并释放内部 Stream 对象。

每个串口只能打开一次，若要再次调用串口时需要关闭已经打开的串口。
